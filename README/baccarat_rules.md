A	1

핸드 점수 계산

모든 카드 점수를 합산

합계의 일의 자리만 사용

7 + 6 = 13 → 3

9 + 8 = 17 → 7

10 + K = 0 → 0

4. 4덱 / 6덱 / 8덱 차이 설명
공통점

👉 확률 분포

카지노 수익 안정화

덱이 많아질수록, 게임은 차가워진다.
